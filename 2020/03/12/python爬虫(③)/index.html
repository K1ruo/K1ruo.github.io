bs对象的find_all方法(name,attrs,recursive,string,**kwargs)

  • name 标签名检索字符串，字符或列表
  • attrs：标签属性值的检索，find_all(‘p’,’course’)检索 p标签，或不设置标签
  • recursive: 是否对子孙全部检索，默认为true
  • string ：检索字符串
