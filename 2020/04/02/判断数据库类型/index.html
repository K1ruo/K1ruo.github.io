判断数据库类型

根据常用搭配

  • asp:access,mssql
  • .net:mssql
  • php:postgresql,mysql
  • jsp:oracle,mysql

根据数据库的标志性信息

  • mssql: select@@version
  • oracle:select banner from v$version
  • mysql: select @@version,version(), length(user)>0
  • postgresql: select version()

根据数据库特有的函数

  • mssql: @@pack_received,@@rowcount
  • mysql:connection_id() last_insert_id() row_count()
  • oracle:bitand(1,1)
  • postgresql: select extract(dom from now())

对字符串的处理方式

  • mssql: select ‘a’+’b’=’ab’
  • mysql:select ‘a’+’b’=’a’’b’,select ‘ab’=concat(‘a’,’b’)
  • oracle:select ‘a’+’b’=’a’||’b’, select ‘ab’=concat(‘a’,’b’)
  • postgresql: select ‘a’+’b’=’a’||’b’, select ‘ab’=concat(‘a’,’b’)

特殊符号

  • mysql中的/**/
  • –是oracle和mssql支持的注释
  • ;是子查询标识符，oracle不支持

对mssql和access数据库的判断

and exist(select count() from sysobjects)>0 mssql
and exist(select count() from msysobjects)>0 accessde
